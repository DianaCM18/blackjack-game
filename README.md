# Juego Blackjack

## Descripción 
Apliocación web que simula el juego de Blackjack.
El jugador puede pedir cartas o plantarse , mientras el sistema calcula automaticamente la suma y determina el resultado de la partida.

## Funcionalidades
- Pedir cartas
- Plantarse
- Calculo automático de la suma de cartas
- Control de turnos
- Interfaz visual interactiva

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
